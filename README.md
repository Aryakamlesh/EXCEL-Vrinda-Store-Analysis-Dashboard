Project Title/Headline --
Vrinda Store Analysis Dashboard

Introduction/Purpose --
The Vrinda Store dashboard is a data analysis project annual sales, revealing women (30-49 yrs) in Maharashtra, Karnataka, and UP are key customers, buying via platforms like Amazon/Flipkart, with peak sales; it helps target marketing efforts using slicers for gender, age, and state. Women (around 64-65%) are primary buyers, with the 30-49 age group contributing the most sales.Top Regions: Maharashtra, Karnataka, and Uttar Pradesh are the leading states for sales.Top Channels: Amazon, Flipkart, and Myntra drive maximum sales.The Vrinda Store wants to create an annual sales report for 2022. So that, the owner of the Vrinda store can understand their customers and grow more sales in 2023.
